ucd-1: Charging

The device battery is re-charged.

Pre-conditions

  • Device has a partially depleted battery

Main event flow

  1. User connects device
  2. The device charge indicator blinks to indicate charging
  3. The device charges

Post-conditions

  • Device has a charged battery

Alternate flow

  • The user disconnects the device from the charger before charging is complete

Linked requirements

  • [org-1] Charging: A user should be able to quickly charge the device with a cell phone charger.
  • [sys-2] Charge indicator location:The charge indicator shall be located 1.0 +/- 0.2 cm from the charging port.
